Raspberry Pi で Mathematica を使おう
Using Mathematica on Raspberry Pi

（平成27年4月27日 更新）
  • 安価なシングルボードコンピュータ Raspberry Pi (wikipedia) に，標準OSである Raspbian をインストールすると，数式処理ソフト Mathematica (wikipedia) を無償で使用することができます．
  • 本ページは，Raspberry Pi に OS をインストールし，無線ネットワークにつなげて Mac からリモートでMathematica を利用できるようにするまで備忘録です．

用意したもの

  • Raspberry Pi 2 Model B
  • 液晶モニター EIZO FlexScan EV2336W-Z
  • USB キーボード Apple Keyboard (テンキー付き - US)
  • USB マウス Apple Mouse
  • Bluetooth Ver.3.0＋HS & Wi-Fi対応 MicroサイズUSBコンボアダプタ PLANEX BT-Micro3H2X
  • micro SD カード (SanDisk microSDXCカード SDSDQUAN-064G-G4A)
  • DVI--HDMI ケーブル Amazonベーシック SK231 （上記の液晶モニターには HDMI 端子がないため，DVI に変換して接続）
  • microUSBケーブル（電源用）
  • 作業用コンピュータ：MacBook Pro Retina, 13-inch, Late 2013 (OS X Yosemite 10.10.3)

OS (Raspbian) のインストール

  • Raspberry Pi財団のWebサイトからOSイメージ (Raspbian) をダウンロードして解凍．
  • 公式 WEB サイトの手順（注意：Mac OS Xの場合）に従って，OSイメージをSDカードへ書き込む．
    主な手順は以下；
    1. SD カードを Mac にさして，FAT32 でフォーマット．
    2. ターミナルから
      $ df -h
      を実行し，SD カードのディバイス名を確認（例：/dev/disk2s1 ）．
    3. SD カードをアンマウント
      $ sudo diskutil unmount /dev/disk2s1
    4. SD カードにOSイメージを書き込む．
      $ sudo dd bs=1m if="イメージファイル名" of=/dev/rdisk2
      注意：ディバイス名が "disk2s1" の場合，先頭に「r」をつけ，「s1」を削除する．このページの Note: Some users have reported issues with using Mac OS X to create SD cards. 以降を参照
    5. 書き込みが終わったら，
      $ sudo diskutil eject /dev/disk2s1
      として，SDカードを取り出す．

起動とその後

  • microSD カードを Raspberry Pi にさす．キーボード，マウス，モニター，ネットワークケーブルを Raspberry Pi に接続し，最後に microUSBケーブルで電源につなぐ．
  • 起動すると，初期設定 (raspi-config) が始まる．expand_rootfs, configue_keyboard, Change_pass, Change_locale などを設定．
  • ユーザーネーム "pi" でログイン（初期パスワードは "raspberry"）．
  • パッケージをアップデートする；
    $ sudo apt-get update
    $ sudo apt-get upgrade
    プロクシ経由の場合は
    $ sudo apt-get -o Acquire::http::proxy="http://(プロキシIP):(ポート番号)" update
    $ sudo apt-get -o Acquire::http::proxy="http://(プロキシIP):(ポート番号)" upgrade
  • X を起動．
    $ startx
